# Trading Bot - Deriv API

Un bot de trading automatisé pour les indices de volatilité Deriv avec gestion des risques.

## Fonctionnalités

- Stratégie multi-indicateurs (EMA, MACD, RSI, Bollinger Bands)
- Gestion des risques avec Take Profit et Stop Loss
- Changement automatique de marché
- Connexion API réelle avec Deriv
- Interface utilisateur intuitive

## Utilisation

1. Connectez-vous avec votre App ID et Token API Deriv
2. Configurez vos paramètres de trading
3. Démarrez le bot
